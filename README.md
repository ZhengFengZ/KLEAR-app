KLEAR is a Java-based power tool that allows you to load data, understand it,
clean it up, reconcile it, and augment it with data coming from
the web. All from a web browser and the comfort and privacy of your own computer.

Official website: **https://klear.org**

Community forum: **https://forum.klear.org**

## Snapshot releases

You can download snapshots of the development version of KLEAR.
To do so, you need to be logged in to GitHub. Then, click on the first item with a green tick / check mark on [this page](https://github.com/KLEAR/KLEAR/actions/workflows/snapshot_release.yml) and scroll down to the Artifacts section to find the version that matches your operating system.

## Run from source

If you have cloned this repository to your computer, you can run KLEAR with:

* `./refine` on Mac OS and Linux
* `refine.bat` on Windows

This requires [JDK 11](https://adoptium.net/) or newer, [Apache Maven](https://maven.apache.org/) and [Node.js 18](https://nodejs.org/) or newer.
